# Crypto Analyzer

Инструмент для консольного сбора и анализа рыночных данных с **CoinGecko** и **CoinMarketCap**.

## Архитектура
Проект реализован по принципу разделения ответственности (**Separation of Concerns**):

* **Infrastructure**: `ApiClient` (сетевое взаимодействие, логика Retry) и `Visualizers` (вывод в Console, JSON, CSV).
* **Parser & Models**: Преобразование сырого JSON в объекты моделей `Cryptocurrency`.
* **Logic**: Провайдеры (оркестрация получения данных) и Анализатор (фильтрация, сортировка, расчет метрик).
* **Entry Point**: `main.py` — CLI-интерфейс на базе Typer.

## Принципы обработки данных (Data Flow)
Поток данных организован линейно:

1. **Инициализация**: Создание компонентов через фабрики.
2. **Запрос**: Получение JSON через `ApiClient`.
3. **Трансформация**: Парсинг в список объектов моделей.
4. **Анализ**: Математический расчет и формирование результирующего словаря.
5. **Экспорт**: Вывод данных выбранным визуализатором.

## Тестирование
Покрытие кода тестами — **98%**.

**Объекты тестирования:**
* **Алгоритмы**: Сортировка и расчеты в `CryptoAnalyzer`.
* **Отказоустойчивость**: Обработка сетевых исключений и лимитов API в декораторе `retry`.
* **Валидация**: Парсинг пустых ответов и специфических структур (CMC `data` key).
* **Интерфейсы**: Проверка полиморфизма визуализаторов через мокирование системных вызовов (`builtins.open`, `rich`).
